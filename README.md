# Tool-convert-GLTF-GLB
The tool allows users to convert GLTF to GLB and vice versa
## Installation
### 1. Clone source
```bash
git clone <repo>
```
### 1. Setup Backend
```bash
cd backend
npm i
npm start
```
### 2. Setup Frontend
```bash
cd frontend/Tool3D
npm i
npm run serve
```
### 3. Import data into Mysql DB
Get file Db.sql and import into Mysql DB
